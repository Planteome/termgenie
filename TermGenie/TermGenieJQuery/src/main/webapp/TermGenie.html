Step 1: Target Ontology

The first step is to select the ontology for which the terms shall be generated.

Step 2: Templates

Once you have selected the ontology, the available term generation patterns can be selected from a menu.

Step 3: Review and Submit

The submit executes a verification of the given input for the selected term generation patterns.
Note: To commit the terms to the ontology a valid user name and password are required.
