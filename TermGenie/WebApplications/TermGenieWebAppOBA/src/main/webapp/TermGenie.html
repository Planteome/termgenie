TermGenie

This TermGenie instance is a tool for creating new terms for OBA. It uses a pattern-based approach to rapidly generate new terms and place them appropriately within the ontology structure. All terms are reviewed by a senior editor before the final commit to the ontology.

Step 1: Ontology Status

The first step is to check the ontology status. If there are issue, such as missing imports or unsatisfiable classes, TermGenie won't/cannot create safely new classes.

Step 2: Templates

Once you have selected the ontology, the available term generation patterns can be selected from a menu.
After selecting and filling templates, click on the 'Verify Input'-Button below to start the next step.

Step 3: Review and Submit

Note: To submit the terms for review to the ontology a valid user name and password are required.
If 'Submit for Review' is disabled the generated terms are not commited and formated for export in the following step.
